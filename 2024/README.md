## 2 · Usage

### Prerequisites

```bash
ocaml --version   # 5.4.1+
```

### Run a solution

Run directly with 
```bash
ocaml day_XX.ml
```

Or compile and run with 
```bash
ocamlopt -o day_XX day_XX.ml
./day_XX
```

where XX is replaced by the number of the day 


> **Note:** Puzzle inputs are personal to each AoC account and are not committed to this repo. Add your own `input_day_XX.txt` to the folder before running a compiled solution
