# 🎄 Advent of Code — My Solutions


[Advent of Code](https://adventofcode.com/) is an annual programming challenge that runs every December. 25 days, 25 puzzles, 2 parts each — a mix of algorithms, data structures, and creative problem-solving.

This repository contains my personal solutions. Feel free to explore or compare approaches !

> **Note:** Following Advent of Code licensing, puzzle inputs are personal to each AoC account and are not committed to this repo. I provide a README each year where you can find how to format yours.
